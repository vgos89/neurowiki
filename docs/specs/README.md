# Section Specs

One spec file per section of the app. Each spec is the single source of truth for that section's design, behavior, and SEO.

## Files
- CALCULATOR_SPEC.md — all 9 calculators
- PATHWAY_SPEC.md — stroke, status epilepticus, migraine, extended IVT, etc.
- TRIALS_SPEC.md — trial database, trial detail pages, trial agent
- ARTICLE_SPEC.md — guide pages, residency toolkit, clinical pearl blocks

## Mockups
Each spec has an accompanying HTML mockup in docs/specs/mockups/. Open in a browser to view.

## Rules
- Specs are locked once approved by PM.
- Changes append to the spec's own changelog at the bottom of the file.
- Every swarm in a section must cite spec line numbers or section IDs in pre-flight.
