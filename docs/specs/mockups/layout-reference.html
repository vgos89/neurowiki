Archetype 3

Width zones — opt-in utilities

Layout does NOT enforce a max-width. Pages opt into a zone via .zone-reading (768px) for long-form or .zone-reference (1024px) for hubs/Home.

.zone-reading — max-w-3xl (768px) · trial detail / calc detail / guide article
Reading zone
Long-form content. Trial detail, calculator detail, guide article, question detail. 768px max width keeps the eye on a comfortable line length.
.zone-reference — max-w-5xl (1024px) · all 4 hub pages, Home
Reference zone
Scanning content with parallel rows. Trials hub, Calc hub, Pathway hub, Guide hub, Home. 1024px allows wider grids without pushing rows past comfortable scan width.
