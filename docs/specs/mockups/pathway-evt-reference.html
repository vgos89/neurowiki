Frame 6 · Cascade-clear demonstration — LVO Location changed Anterior → Basilar
What this frame demonstrates: The user has just changed LVO Location from Anterior to Basilar in Step 1. The pathway has automatically cleared Steps 2–4 (because those answers no longer apply on the basilar branch). The transient notice immediately below the changed row tells the user what happened and offers Undo. The notice auto-dismisses after 4 seconds. The Undo button restores the prior LVO Location value AND all the downstream answers that were wiped — one tap reverses the cascade.
